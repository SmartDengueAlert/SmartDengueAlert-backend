# Dengue Prediction App

## Setup

### 1. Install Dependencies

```bash
npm install
```
### 2. Initialize the Database

```bash
npm run init-db
```
### 3. Start the Server

3. Start the Server

API Endpoints
Register a new user
POST /auth/register

User login
POST /auth/login

Make a prediction
POST /predictions

Get weather data
GET /weather

### Step 5: Run the Initialization Script

Now, whenever you or someone else needs to set up the database, they can simply run:

```bash
npm run init-db
```
